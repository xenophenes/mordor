how can I contribute?

If you are interested in supporting us, the best way is to become a member and actively contribute to our mission.

However, donations are also an excellent way to express appreciation for what we do.

website

This website is managed using github.

You can view the revision history via gitweb and send any updates, either as a git pull request or a patch, to webmaster@pgus.org.

postgresql

PostgreSQL.org is the home of the global PostgreSQL community.

If you are looking for community support or software downloads you should visit the global site. We also encourage curiosity for curiosity's sake!
